# Cheat Sheet
## Set
1. Initialization:
    ```javascript
    const set = new Set();
    ```
2. length:
    ```javascript
    set.size;
    ```
3. add:
    ```javascript
    set.add(ele);
    ```
4. delete:
    ```javascript
    set.delete(ele);
    ```
5. check validation of a certain element
    ```javascript
    set.has(ele);
    ```
